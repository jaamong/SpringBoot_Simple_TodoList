<h3> :hammer: 사용 기술</h3>

* Java 17
* SpringBoot 3.1.0
* Gradle
* Spring Web
* Lombok
* Thymeleaf

<br><br>

<h3> :page_with_curl: URL과 기능</h3>

| URL               | 기능                |
|-------------------|-------------------|
| /todo/home        | 기본 화면, todo 목록 조회 |
| /todo/create      | 새로운 todo 등록       |
| /todo/{id}/update | todo 상태 변경        |
| /todo/{id}/delete | 등록한 todo 삭제       |
